## Known Issues

1. **Devnet faucet**: `requestAirdrop` has 2-request/8-hour limit per IP. Tests fund wallets from deployer via transfer.
2. **Dispute window**: 24h on-chain. `finalize_result_after_dispute_window` tested on localnet only (clock warp).
3. **IDL**: Generated by `npm run generate:idl`, not `anchor build`. See TOOLCHAIN.md.
